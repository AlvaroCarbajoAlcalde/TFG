Instrucciones:
Para el funcionamiento del programa en Windows, es necesario instalar AccessDatabaseEngine, para ver la solución hay que instalar VisualStudio 2019.
Para poder hacer funcionar la aplicacion móvil, es necesario tener un dispositivo físico de Android. Puedes copiar en él el archivo .apk e instalarlo. Recuerda habilitar la instalación de aplicaciones externas a googlePlay en tu dispositivo.
Para conectar ambos programas, los dispositivos deben estar conectados a la misma red local LAN.
